## Overview

A sophisticated, responsive website for The Lola Group, a premier real estate investment banking firm. The platform showcases comprehensive capital advisory services, development financing solutions, and the firm's extensive global expertise in complex real estate transactions.

## Features

- **Professional Navigation** - Intuitive navigation menu with smooth scrolling to key sections
- **Responsive Design** - Mobile-first approach using Bootstrap 5 for optimal viewing on all devices
- **Dynamic Interactions** - JavaScript-powered interactive elements for enhanced user engagement
- **Video Integration** - Striking hero section with autoplay video background
- **Service Showcase** - Detailed presentation of core services including development finance and structured financing
- **Premium Styling** - Modern dark theme with goldenrod accents for a sophisticated aesthetic

## Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Advanced styling and responsive layouts
- **JavaScript** - Vanilla JS for interactivity and navigation
- **Bootstrap 5** - Frontend framework for responsive components
- **Video Background** - Multimedia content integration

## Installation & Setup

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd lola-group-website
   ```

3. Open `index.html` in your web browser or serve using a local server:
   ```bash
   python -m http.server 8000
   ```
